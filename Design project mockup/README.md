# SmartSlot - Queue Booking & Last-Minute Deals Platform

**Tagline:** Skip Waiting. Save Money. Book Smart.

## Overview

SmartSlot is a modern web application that solves two major problems:
1. **For Users**: Eliminates waiting time through queue booking and offers last-minute deals
2. **For Businesses**: Fills empty time slots and manages customer flow digitally

## Features

### User Features
- 🗺️ **Location-based Discovery** - Find nearby services instantly
- ⏰ **Queue Management** - Book slots or get queue tokens
- 💰 **Last-Minute Deals** - Save up to 30% on empty time slots
- 📅 **Booking Management** - Track and manage all your bookings
- ⭐ **Ratings & Reviews** - Make informed decisions
- 🎯 **Smart Recommendations** - AI-powered best time to visit predictions

### Business Features
- 📊 **Analytics Dashboard** - Track bookings, revenue, and trends
- 🎫 **Slot Management** - Create and manage time slots
- 💸 **Deal Creation** - Offer discounts on empty slots
- 📈 **Revenue Insights** - Monitor business performance
- 👥 **Customer Management** - View and manage bookings

## Technology Stack

- **Frontend**: React 18 + TypeScript
- **Routing**: React Router v7
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion)
- **Charts**: Recharts
- **Icons**: Lucide React
- **UI Components**: Radix UI
- **Notifications**: Sonner

## Pages

1. **Home** - Hero section, features, categories, and call-to-action
2. **Services** - Browse and search available services with filters
3. **Service Detail** - View details, book slots, or join queue
4. **Deals** - Last-minute discounted offers
5. **My Bookings** - User's booking history and management
6. **Business Dashboard** - Analytics and management for businesses

## Key Design Decisions

- **Modern Gradient Design**: Blue to purple gradients for a premium feel
- **Responsive Layout**: Works seamlessly on desktop and mobile
- **Real-time Indicators**: Color-coded wait times (green/yellow/red)
- **Intuitive UX**: Clear CTAs and smooth navigation
- **Data Visualization**: Charts for business analytics

## Getting Started

1. Navigate through the home page to explore features
2. Click "Explore Services" to browse nearby services
3. Select a service to view details and book
4. Check "Deals" for last-minute discounted slots
5. View all bookings in "My Bookings"
6. Business owners can access the dashboard via "For Business"
